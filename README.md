# 🍽️ Recipe Journal

Recipe Journal is a simple multi-page recipe website built using **HTML only**. It features a collection of delicious recipes from different cuisines, making it a beginner-friendly web development project.

## 🌐 Live Website

🔗 https://dineshwaria.github.io/recipe_journal/

## 📖 Features

- 🥞 Breakfast recipe
- 🍝 Main course recipes
- 🍰 Dessert recipes
- 🥤 Beverage recipe
- 📋 Ingredients list
- 👨‍🍳 Step-by-step cooking instructions
- ⏱️ Preparation time table
- 💡 Helpful cooking tips
- 🔗 Navigation between pages

## 🍽️ Included Recipes

- Pancakes
- White Sauce Pasta
- Margherita Pizza
- Ramen
- Chocolate Cake
- Chocolate Chip Cookies
- Lassi

## 🛠️ Technologies Used

- HTML5
- GitHub Pages
